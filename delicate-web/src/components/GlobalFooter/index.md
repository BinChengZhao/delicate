页脚属于全局导航的一部分，作为对顶部导航的补充，通过传递数据控制展示内容。

## API

| 参数      | 说明     | 类型                                                             | 默认值 |
| --------- | -------- | ---------------------------------------------------------------- | ------ |
| links     | 链接数据 | array<{ title: ReactNode, href: string, blankTarget?: boolean }> | -      |
| copyright | 版权信息 | ReactNode                                                        | -      |
